if-else

“if else” statement: executes one block of code if a condition is met and a different block of code if it is not met.

light <- "red"

if (light == "green") {
  saber <- "Jedi"
} else {
  saber <- "Sith"
}

saber
[1] "Sith"

If-else

Generic blueprint of if-else in R:

if (condition) {
  # what to do if condition is TRUE
} else {
  # what to do if condition is FALSE
}

Simplified If

If we only care about the if part:

if (condition) {
  # what to do if condition is TRUE
}

Nested if’s

You can chain multiple if-else’s

light <- "purple"

if (light == "green") {
  saber <- "Jedi"
} else if (light == "red") {
  saber <- "Sith"
} else {
  saber <- "Dathomir"
}

saber
[1] "Dathomir"

Question

Given an object year <- sample(1:4, size = 1), write a decision tree in R using nested if-else statements to determine its corresponding name

02:00 

year <- sample(1:4, size = 1)

if (year == 1) {
  name <- "freshman"
} else if (year == 2) {
  name <- "sophomore"
} else if (year == 3) {
  name <- "junior"
} else {
  name <- "senior"
}

name
[1] "freshman"
