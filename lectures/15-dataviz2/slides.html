ggplot() syntax

ggplot() builds a plot layer by layer, each one added on top of one another with + (not |>).

  • ggplot() creates canvas
  • aes() creates mappings, called inside ggplot() or a geom()
  • geom_() puts down marks using declared geometry
  • Usually you specify the data in ggplot()
  • the data must be a dataframe or tibble

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, 
                     y = flipper_length_mm))

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, 
                     y = flipper_length_mm)) +
  geom_point()

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, 
                     y = flipper_length_mm,
                     color = species)) +
  geom_point()

More on Mappings

Recap

  • An aesthetic mapping links a variable in the data to a visual attribute.
  • ggplot() and geom_() have a mapping argument.
  • We use the aes() to specify such mappings, e.g.
    • ggplot(..., aes(x = species))
    • geom_bar(aes(x = species))

Global -vs- Local Mappings

We distinguish 2 types of aesthetic mappings:

  • Global Mappings: those specified inside ggplot()


  • Local Mappings: those specified inside geom_() functions

# global mappings
ggplot(data = penguins,
       aes(x = bill_length_mm, 
           y = flipper_length_mm)) +
  geom_point()
# local mappings
ggplot(data = penguins) +
  geom_point(
    aes(x = bill_length_mm, 
        y = flipper_length_mm))  

# global mappings
ggplot(data = penguins,
       aes(x = bill_length_mm, 
           y = flipper_length_mm)) +
  geom_point() +
  geom_smooth(method = "lm")
# local mappings (be careful!!!)
ggplot(data = penguins) +
  geom_point(
    aes(x = bill_length_mm, 
        y = flipper_length_mm))  +
  geom_smooth(method = "lm")

labs() allows you to add

  title = "...",
  subtitle = "...",
  x = "...",
  y = "..."
)

       mapping = aes(x = bill_length_mm, 
                     y = flipper_length_mm,

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, 
                     y = flipper_length_mm,
                     color = species)) +
  geom_point() +
  labs(
    title = "Bill and flipper length",
    subtitle = "Positive relation",
    x = "Bill length (mm)",
    y = "Flipper length (mm)")

Facets

Facets

  • facet_wrap() wraps a 1d sequence of panels into 2d

  • facet_grid() forms a matrix of panels defined by row and column faceting variables

    • facet_grid(rows = vars(...))
    • facet_grid(cols = vars(...))
    • facet_grid(row_var ~ col_var)

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, y = flipper_length_mm, color = species)) +
  geom_point() +
  facet_wrap(~ species)

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, y = flipper_length_mm, color = species)) +
  geom_point() +
  facet_wrap(~ species, nrow = 2)

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, y = flipper_length_mm, color = species)) +
  geom_point() +
  facet_grid(. ~ species)  # facet_grid(cols = vars(species))

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, y = flipper_length_mm, color = species)) +
  geom_point() +
  facet_grid(species ~ .)  # facet_grid(rows = vars(species))

ggplot(data = penguins,
       mapping = aes(x = bill_length_mm, y = flipper_length_mm, color = species)) +
  geom_point() +
  facet_grid(species ~ island)

Because the mappings for x, y and color are the same in both calls to geom_point()

  • x = bill_length_mm
  • y = flipper_length_mm
  • color = species

we can simplify the command in the previous slides as follows (next slide)

ggplot(
  data = filter(penguins, sex == "male"),
  aes(x = bill_length_mm,
      y = flipper_length_mm,
      color = species)) +
  geom_point(size = 2, alpha = 0.5) +
  geom_point(
    data = big_males,
    size = 4) +
  theme_minimal()
