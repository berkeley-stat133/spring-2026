Vectors

  • Central data structure in R

  • Ordered set of elements

  • Atomic: elements of the same type: typeof()

  • Have length()

  • Elements can be named: names()

Atomic*

Special values in an atomic vector.

  • NA: Not Available / Missing

  • NULL: Does not exist (nothing)

  • NaN: Not a Number (e.g. dividing by zero)

  • Inf: Infinity

Functions to create vectors

  • c()
  • vector(mode = , length = )
    • logical(), integer(), double(), character()
  • seq(from = , to = , by = )
  • rep(x, each = , times = , length.out = )
  • colon operator: 1:10

Vectorization

Vectorized: Operations and functions that can run on an entire vector at once (i.e. element-wise computations)

dbl_vec <- c(1, 2.3, 4)

log(dbl_vec) # vectorized
[1] 0.0000000 0.8329091 1.3862944


# not vectorized
sum(dbl_vec) # vector-as-argument
[1] 7.3

Vector Recycling

Vector Recycling: If a function requires a longer vector than provided, it will fill the vector with the existing elements, in order.

c(2, 6, 1)^2
[1]  4 36  1

Coercion

Implicit Coercion: Automatic change of data types to ensure atomic vectors.

c(1, "dog", TRUE)
[1] "1"    "dog"  "TRUE"


Changes follow a hierarchy.

logical < integer < double < character

Coercion

Explicit Coercion: R provides functions to explicitly coerce from one type to another.

  • as.logical()

  • as.integer()

  • as.double()

  • as.character()

Depending on what data-type you want to coerce to, these functions may or may not work.

as.double(c(TRUE, FALSE, FALSE, FALSE))
[1] 1 0 0 0

Vector Subsetting

Square Bracket Syntax

Subsetting by Index

aka numeric subsetting

char_vec <- c("bear", "owl", "squirrel")
char_vec
[1] "bear"     "owl"      "squirrel"
char_vec[2]
[1] "owl"
char_vec[2:3] 
[1] "owl"      "squirrel"
char_vec[-2] 
[1] "bear"     "squirrel"


Recall that R starts indexing at 1!

Subsetting by Name

aka character subsetting

char_vec2 <- c("oski" = "bear", "fiat" = "owl", "lux" = "squirrel")
char_vec2
      oski       fiat        lux 
    "bear"      "owl" "squirrel" 
char_vec2["oski"]
  oski 
"bear" 
char_vec2[c("lux", "fiat")]
       lux       fiat 
"squirrel"      "owl"

Subsetting by Logicals

aka logical subsetting

char_vec2
      oski       fiat        lux 
    "bear"      "owl" "squirrel" 


char_vec2[c(TRUE, FALSE, TRUE)]
      oski        lux 
    "bear" "squirrel"

Note: the logical vector should be of the same length.

char_vec2[TRUE]
      oski       fiat        lux 
    "bear"      "owl" "squirrel"

Change in place syntax

You can assign into a subsetted vector to change its contents in place.

z <- c(latte = 2, boba = 4, matcha = 6)
z[c(TRUE, FALSE, TRUE)] <- 12
z
 latte   boba matcha 
    12      4     12 
names(z)
[1] "latte"  "boba"   "matcha"
names(z) <- c("A", "B", "C")
z
 A  B  C 
12  4 12
