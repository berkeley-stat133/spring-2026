Quarto Pub

Publishing Steps

Assume I have a file called demo-pub2.qmd opened in Positron. To publish my qmd file to my QuartoPub account I follow these steps:

  1. Go to the Terminal tab.
    • Make sure your working directory contains the qmd file.
  2. Type command: quarto publish demo-pub2.qmd
  3. Choose provider Quarto Pub and press return (enter)
  4. Publish with default account (linked to my email address)
  5. Optionally: you can change the name of the document
  6. Press return (enter)

. . .

Once published, there will be an associated link (URL), something like: https://username.quarto.pub/demo-pub-2

Submission to bCourses
