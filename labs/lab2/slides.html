Terminal and Basic Unix commands

Terminal in IDE

  1. Launch your IDE (Positron or RStudio)

  2. Locate the Terminal tab (next to the Console tab)

  3. Click on the Terminal

  • We use the Terminal to interact with the operating system.

  • The way we carry out this interaction is by typing commands, e.g.

    • pwd: print working directory
    • ls: list files
    • cd: change directory
    • mkdir: make directory

In your Terminal, type the following commands:

. . .

Terminal
# print working directory (in theory your Home directory)
pwd


. . .

Terminal
# list files in current directory

Code Chunks (aka code cells)

```{r}

```


A code chunk is defined by:

  • 3 “opening” consecutive backticks
  • immediately followed by curly braces
    • r inside curly braces (R code cell)
  • 3 “closing” consecutive backticks
  • You write R code inside a chunk

R Code Chunks

```{r}
2 + 3
```


```{r}
name <- c("Leia", "Luke", "Han")
```


```{r}
seq(from = 1, to = 10, by = 2)
```

Keyboard shortcuts to insert a code cell

Positron:

  • Mac: Cmd + Shift + I
  • Windows: Crtl + Shift + I

RStudio:

  • Mac: Cmd + Option + I
  • Windows: Crtl + Alt + I

. . .

  • Don’t be a code chunk miser (stingy)
  • Avoid “monster” chunks with too many things in them
  • Better to use simpler chunks
    • easier to understand
    • easier to inspect
    • easier to debug

Publishing Steps

Assume I have a file called demo-pub1.qmd opened in Positron. To publish my qmd file to my QuartoPub account I follow these steps:

  1. Go to the Terminal tab.
    • Make sure your working directory contains the qmd file.
  2. Type command: quarto publish demo-pub1.qmd
  3. Choose provider Quarto Pub and press return (enter)
  4. Publish with default account (linked to my email address)
  5. Optionally: I can change the name of the document
  6. Press return (enter)

. . .

Once published, there will be an associated link (URL), something like: https://username.quarto.pub/demo-pub-1
